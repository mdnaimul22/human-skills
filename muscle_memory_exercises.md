# 🧠 পাইথন মাসল মেমোরি: টাইপ করে শেখার প্র্যাকটিস

প্রোগ্রামিং শেখার সবচেয়ে ভালো উপায় হলো **"Muscle Memory"** বা হাত দিয়ে টাইপ করে ব্রেনে গেঁথে ফেলা। এই ডকুমেন্টের কোডগুলো আপনি **কখনোই কপি-পেস্ট করবেন না**। দেখে দেখে নিজের কিবোর্ডে টাইপ করবেন। একবার টাইপ করলেই আপনি ফিল করতে পারবেন কোডটি কীভাবে কাজ করছে!

---

## 💻 প্র্যাকটিস ১: ডিসিশন মেকিং (If-else)
**উদ্দেশ্য:** পাইথনে কীভাবে শর্ত দিয়ে কোড লিখতে হয় তা শেখা।
**আপনার কাজ:** নিচের কোডটি আপনার কম্পিউটারে (যেকোনো `.py` ফাইলে) হুবহু টাইপ করুন এবং রান করুন।

```python
# ১. ইউজারের তথ্য সেট করা
user_age = 22
has_ticket = True

# ২. লজিক চেক করা
if user_age >= 18 and has_ticket == True:
    print("✅ স্বাগতম! আপনি মুভিটি দেখতে পারবেন।")
elif user_age >= 18 and has_ticket == False:
    print("❌ আপনার বয়স ঠিক আছে, কিন্তু টিকিট নেই। টিকিট কিনে আসুন।")
else:
    print("⛔ দুঃখিত! এই মুভিটি শুধুমাত্র ১৮+ বয়স্কদের জন্য।")
```

**🔥 ছোট চ্যালেঞ্জ:** টাইপ করার পর `user_age` এর মান পরিবর্তন করে `15` দিন এবং কোড আবার রান করে দেখুন কী প্রিন্ট হয়।

## 💻 প্র্যাকটিস ২: অটোমেশন এবং লুপ (For Loop)
**উদ্দেশ্য:** একটি লিস্টের ভেতর দিয়ে ঘুরে ঘুরে বারবার একই কাজ করা।
**আপনার কাজ:** নিচের কোডটি হুবহু টাইপ করুন। খেয়াল রাখবেন `for` এর ভেতরের লাইনগুলোতে যেন স্পেস (Indentation) ঠিক থাকে।

```python
# ১. কাস্টমারদের একটি লিস্ট তৈরি
customers = ["নাঈমুল", "রাফসান", "আহমেদ", "সাকিব"]

# ২. একটি লুপ চালানো যা সবার কাছে মেসেজ পাঠাবে
print("--- মেসেজ সেন্ডিং শুরু ---")

for name in customers:
    print(f"📧 সেন্ডিং ইমেইল টু: {name}...")
    print(f"   মেসেজ: হ্যালো {name}, আমাদের নতুন অফার চেক করুন!\n")

print("--- মেসেজ সেন্ডিং শেষ ---")
```

**🔥 ছোট চ্যালেঞ্জ:** টাইপ করার পর লিস্টে আপনার নিজের নাম ও আরও দুজন বন্ধুর নাম যোগ করে লুপটি রান করুন।

## 💻 প্র্যাকটিস ৩: নিজস্ব মেশিন বা ফাংশন তৈরি (Functions)
**উদ্দেশ্য:** কোডকে একটি প্যাকেটে বা ব্লকে বন্দি করা, যাতে বারবার একই কোড লিখতে না হয়।
**আপনার কাজ:** হুবহু টাইপ করুন।

```python
# ১. একটি ক্যালকুলেটর ফাংশন তৈরি করা
def calculate_discount(price, discount_percent):
    discount_amount = price * (discount_percent / 100)
    final_price = price - discount_amount
    return final_price

# ২. ফাংশনটিকে ব্যবহার করা (Call করা)
laptop_price = 100000
phone_price = 45000

# ১০% ডিসকাউন্টে ল্যাপটপের দাম
laptop_final = calculate_discount(laptop_price, 10)
print(f"ল্যাপটপের আসল দাম {laptop_price} টাকা, ডিসকাউন্টের পর: {laptop_final} টাকা")

# ৫% ডিসকাউন্টে ফোনের দাম
phone_final = calculate_discount(phone_price, 5)
print(f"ফোনের আসল দাম {phone_price} টাকা, ডিসকাউন্টের পর: {phone_final} টাকা")
```

**🔥 ছোট চ্যালেঞ্জ:** টাইপ করা শেষ হলে, নিজে থেকে একটি স্মার্টওয়াচ (Smartwatch) এর দাম সেট করে তার উপর ২০% ডিসকাউন্ট হিসাব করে প্রিন্ট করুন।

### 📝 কেন এই মেথডটি কাজ করে?
আপনি যখন হাত দিয়ে `def`, `if`, বা `for` টাইপ করবেন, তখন আপনার ব্রেন অটোমেটিক্যালি এর সিনট্যাক্স এবং লজিক মনে রাখতে শুরু করবে। এভাবেই পৃথিবীর সব সেরা প্রোগ্রামাররা কোডিং শিখেছেন! শুভকামনা আপনার টাইপিং প্র্যাকটিসের জন্য! 🚀
